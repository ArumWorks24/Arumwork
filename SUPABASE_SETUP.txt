🎯 SUPABASE SIMPLE SETUP GUIDE (Like I'm 5!)

==========================================
WHAT IS SUPABASE? (Super Easy Explanation!)
==========================================

Imagine Supabase is like a SHARED NOTEBOOK 📓
- Your website writes orders in it
- Your admin panel reads from it
- Everyone can see the same notes!

Without Supabase = Each phone has its own secret notebook (not shared!)
With Supabase = One notebook everyone shares ✅

==========================================
STEP 1: LOGIN TO SUPABASE
==========================================

1. Go to: https://supabase.com
2. Click "Login" 
3. Login with your account (the same account you used to create fmbnplpuitedqlvkddke)

==========================================
STEP 2: FIND YOUR PROJECT
==========================================

1. After login, you'll see your projects
2. Click on the project named something with "fmbnplpuitedqlvkddke"
3. Wait for it to load (may take few seconds)

==========================================
STEP 3: CREATE "ORDERS" TABLE (The Notebook!)
==========================================

Once you're in the project:

1. Look at the LEFT side menu
2. Click on "Table Editor" (icon looks like 📊)
3. Click the + button that says "New table"
4. A new page will open

FILL IN LIKE THIS:

📝 Table name: orders
   (type exactly: orders)

5. Click "Create table" button (blue button at bottom)

==========================================
STEP 4: ADD COLUMNS (The Columns in Notebook!)
==========================================

Now you need to add the parts of an order!
Click "+ Add column" button for EACH of these:

1. Column name: id          Type: text
2. Column name: service     Type: text  
3. Column name: description Type: text
4. Column name: price       Type: number
5. Column name: delivery    Type: text
6. Column name: firstName   Type: text
7. Column name: lastName    Type: text
8. Column name: phone       Type: text
9. Column name: summary    Type: text
10. Column name: status     Type: text
11. Column name: date       Type: text
12. Column name: orderTime  Type: timestamp

For EACH column:
- Type the name
- Select the type from dropdown
- Click Save!

==========================================
STEP 5: MAKE IT PUBLIC (Very Important!)
==========================================

This is like letting anyone write in your notebook!

1. In the left menu, click "Settings" (⚙️ icon)
2. Click "API" 
3. Look for "Row Level Security"
4. If it's ON, turn it OFF
   (Or create a policy that allows everyone to do everything)

EASIER WAY:
1. Go back to your "orders" table
2. Click "Add" Policy button
3. Choose "Full access to everyone"
4. Name it anything (like "Allow all")
5. Click Save!

==========================================
TROUBLESHOOTING (If Not Working!)
==========================================

Problem: "Cloud not available"
Solution: 
- Did you create the table? ✓
- Did you add all 12 columns? ✓
- Did you turn off or allow Row Level Security? ✓

Problem: Can't see the table
Solution: Refresh the page!

Problem: Columns missing
Solution: Go back to table editor, check each column exists!
